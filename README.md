# CashControl
A finance management app is a tool that allows users to effectively
monitor and control their expenses and savings. The app offers features such as
tracking income and expenses.

# Technologies

- Angular
- TypeScript
- RXJS
- NGRX Store
- Angular Material
- SCSS
- NestJS
- MongoDB
- Valibot
